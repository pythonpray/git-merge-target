## 一键将当前分支合并到目标分支并推送到远程仓库,并切回到当前分支,避免来回切换分支的繁琐步骤

自动执行以下操作：

- 自动切换到目标分支
- git pull远端同名分支
- git merge合并当前分支
- 推送到远端同名分支
- 自动切回原分支
